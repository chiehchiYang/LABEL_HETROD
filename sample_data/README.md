# Sample Data

This folder contains a minimal two-entity sample set for GitHub preview and quick smoke tests.

Files:
- 00_background.png
- 00_background_semantic.png
- 00_tracksMeta.csv
- 00_tracks.csv
- area_color_class.json
- labeled_scenarios.json
- pet_distance_dict.json
- track_frame_dict.json
- trackid_class.json
- trackid_objects.json

Use `python prepare_sample_data.py` to copy these files into `data/` for local demo runs.
